## Installation
```bash
git clone git@github.com:lan33/ocr-web-development.git

cd ocr-web-development

python -m venv .venv

# Windows:
.venv\Scripts\activate
# Unix/MacOS:
source .venv/bin/activate

pip install -r requirements.txt
```

## Lancement de l'application
```bash
fastapi dev main.py
```
